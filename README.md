# GUI MAUI - Prosta aplikacja do produktów

To jest prosty projekt w .NET MAUI. Aplikacja pokazuje listę produktów i pozwala je dodawać, edytować, usuwać i zapisywać do pliku CSV.

---

## Funkcje

- Pokazuje listę produktów (nazwa, cena, kategoria)  
- Dodawanie nowego produktu  
- Edytowanie istniejącego produktu  
- Usuwanie produktu  
- Zapis produktów do CSV  
- Wczytywanie produktów z CSV przy starcie aplikacji  

---

## Jak używać

1. Otwórz projekt w Visual Studio z .NET MAUI  
2. Uruchom aplikację (F5)  
3. Lista produktów pojawi się automatycznie, jeśli istnieje plik CSV  
4. Używaj przycisków:  
   - **Dodaj** → nowy produkt  
   - **Edytuj** → zmiana wybranego produktu  
   - **Usuń** → usuwa wybrany produkt  
   - **Zapisz CSV** → zapisuje wszystkie produkty
